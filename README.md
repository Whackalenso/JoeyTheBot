# JoeyTheBot

A Discord moderation bot with loads of features

Prefix is .<command>

## Commands:

Analytics:
  statistics                  
ContentFilter:
  disable                     
  enable                      
ModApplications:
  apply                       
  deleteApplication           
  setQuestions                
  talk                        
  toggleAnonymousApplications 
MoveMessages:
  moveMessage                 
  moveMessages                
Verification:
  setName                     
  verify
